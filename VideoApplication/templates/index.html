Live Video Demonstration

This web application utilizes a Convolutional Neural Network in order to detect whether someone wears a mask or not. Process is divided in two stages: first one face detection algorithm is used in order to extract faces from frames given from video and then the CNN model classifies individual.
This application constitutes a real case on which CNNs can be used. The present application could be a CCTV camera on places where a big number of people is gathered.
Start and Stop button below, initiare and terminate recording. More details for a specific recording can be found on log file.
